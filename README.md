# 🚀 Product Management

A modern product management dashboard built for a Full Stack Internship assessment.

## ✨ Features
- Add, Edit, Delete Products
- Persistent storage using LocalStorage
- 🔍 Search functionality
- 🔔 Toast notifications
- 🌙 Dark/Light mode
- Responsive UI

## 🛠 Tech Stack
- Next.js (App Router)
- Tailwind CSS
- React Hooks

## ⚙️ Setup

```bash
npm install
npm run dev
